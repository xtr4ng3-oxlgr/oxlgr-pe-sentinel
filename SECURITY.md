# Security Policy

OXLGR PE Sentinel es una herramienta defensiva de análisis estático.

## Alcance

Permitido:

- análisis de archivos propios,
- análisis de muestras autorizadas,
- uso educativo,
- triage defensivo,
- revisión en laboratorio,
- generación de reportes.

Fuera de alcance:

- explotación,
- evasión,
- ejecución de muestras,
- modificación de binarios,
- automatización ofensiva,
- clasificación acusatoria sin contexto.

## Reportar problemas

Para reportar un bug del proyecto, abrir un issue con:

- sistema operativo,
- compilador,
- versión de CMake,
- comando usado,
- error completo,
- archivo de prueba si se puede compartir legalmente.

No adjuntar muestras maliciosas reales al repositorio.
